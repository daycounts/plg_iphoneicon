This plugin adds meta tags to be compliant with iPhone/iPad browsing. It adds a home screen icon and a loading image.

For more information on creating icons, see http://developer.apple.com/library/ios/#documentation/UserExperience/Conceptual/MobileHIG/IconsImages/IconsImages.html

For a template to create iPhone/iPad icons, see http://blog.cocoia.com/2010/iphone-4-icon-psd-file/



################################
#   INSTALLATION INSTRUCTIONS  #
################################

1. Login to Joomla administration
2. Install with Joomla Installer
3. Go to plugin manager, filter on system plugins
4. Edit the new plugin System - iPhone icon and activate plugin
5. You're done !
